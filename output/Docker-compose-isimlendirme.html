Docker compose isimlendirme

Thu 13 September 2018
  • compose links'te service parametresini kabul ediyor. container_name parametresine hata veriyor.
  • ps ciktisinda container_name'i donduruyor.

asagidaki compose file ile test ettim;

version: '3'
services:

  service-1:
    container_name: konteyner-1
    image: busybox
    command: tail -f /etc/hosts

  service-2:
    container_name: konteyner-2
    image: busybox
    command: tail -f /etc/hosts
    links:
      - service-1:xyz
